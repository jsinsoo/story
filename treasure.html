10. 보물섬 장면

English
Doctor Livesey came to treat him.

한글 번역
리브지 박사가 그를 치료하러 왔다.

# 10번 문장: 데코레이터
def treasure_frame(func):
    def wrapper():
        print("🏴‍☠️ 보물섬 장면 시작")
        func()
        print("💰 파이썬 학습 완료")
    return wrapper

@treasure_frame
def show_scene():
    print('리브지 박사가 그를 치료하러 왔다.')
    print("→", 'Doctor Livesey came to treat him.')

show_scene()

Jim never forgot his unforgettable adventure on Treasure Island.

# 100번 문장: 데코레이터
def treasure_frame(func):
    def wrapper():
        print("🏴‍☠️ 보물섬 장면 시작")
        func()
        print("💰 파이썬 학습 완료")
    return wrapper

@treasure_frame
def show_scene():
    print('짐은 보물섬에서의 잊을 수 없는 모험을 평생 기억했다.')
    print("→", 'Jim never forgot his unforgettable adventure on Treasure Island.')

show_scene()
