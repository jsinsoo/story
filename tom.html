기초 01 · 클래스와 객체

톰 소여의 일상

톰 소여는 미시시피 강가의 작은 마을에 살고 있었다.

Tom Sawyer lived in a small town along the Mississippi River.

인물 하나를 클래스로 만들고, 객체를 생성해 정보를 출력한다.

class Character:
    pass

tom = Character()
tom.name = "Tom Sawyer"
tom.home = "a small town along the Mississippi River"

print(tom.name)
print(tom.home)
기초 02 · 생성자 __init__

장난꾸러기 소년

톰은 공부보다 모험과 장난을 더 좋아하는 소년이었다.

Tom was a boy who loved adventures and mischief more than studying.

생성자를 사용하면 객체가 만들어질 때 필요한 정보를 바로 저장할 수 있다.

class Boy:
    def __init__(self, name, favorite):
        self.name = name
        self.favorite = favorite

    def introduce(self):
        print(f"{self.name} loves {self.favorite}.")

tom = Boy("Tom", "adventures and mischief")
tom.introduce()
기초 03 · 인스턴스 변수

폴리 이모의 보살핌

그는 폴리 이모의 보살핌을 받으며 자랐다.

He grew up under the care of Aunt Polly.

객체마다 다른 값을 가지는 변수를 인스턴스 변수라고 한다.

class FamilyCare:
    def __init__(self, child, guardian):
        self.child = child
        self.guardian = guardian

care = FamilyCare("Tom", "Aunt Polly")
print(care.child, "is cared for by", care.guardian)
기초 04 · 메서드

꾸중을 듣는 톰

톰은 자주 말썽을 부려 이모에게 꾸중을 들었다.

Tom often got into trouble and was scolded by his aunt.

객체가 할 수 있는 동작을 메서드로 표현한다.

class Tom:
    def make_trouble(self):
        print("Tom made trouble.")

tom = Tom()
tom.make_trouble()

기초 05 · 여러 객체

울타리 칠하기

어느 토요일 아침 톰은 벌로 긴 울타리를 칠해야 했다.

One Saturday morning, Tom was punished by having to paint a long fence.

같은 클래스로 서로 다른 역할을 가진 여러 객체를 만들 수 있다.

class Task:
    def __init__(self, person, job):
        self.person = person
        self.job = job

    def show(self):
        print(f"{self.person} must {self.job}.")

task = Task("Tom", "paint a long fence")
task.show()
기초 06 · 리스트와 객체

친구들의 관심

친구들은 톰이 울타리 칠하기를 재미있어하는 것처럼 보이자 그 일을 해 보고 싶어 했다.

His friends wanted to try painting because Tom made it look enjoyable.

객체 안에 리스트를 넣어 여러 친구를 관리한다.

class FenceScene:

scene = FenceScene()
scene.add_friend("Ben")
scene.add_friend("Billy")
print(scene.friends)
기초 07 · 객체 상태 변경

울타리 완성

결국 톰은 일을 거의 하지 않고도 울타리가 모두 칠해지게 만들었다.

In the end, Tom got the fence painted without doing much work himself.

객체의 상태를 메서드로 바꾸는 방법을 배운다.

class Fence:
    def __init__(self):
        self.painted = 0

    def paint(self, amount):
        self.painted += amount

fence = Fence()
fence.paint(40)
fence.paint(60)
print("painted:", fence.painted, "%")
기초 08 · __str__

베키 대처

어느 날 학교에 베키 대처라는 예쁜 소녀가 전학 왔다.

One day, a pretty girl named Becky Thatcher transferred to his school.

__str__ 메서드는 객체를 보기 좋은 문자열로 표현한다.

class Student:
    def __init__(self, name, school_status):
        self.name = name
        self.school_status = school_status

    def __str__(self):
        return f"{self.name}: {self.school_status}"

becky = Student("Becky Thatcher", "new student")
print(becky)

묘지의 비밀

One night, Tom went to the graveyard with his friend Huckleberry Finn.

응용 11 · property

살인 사건 목격

톰과 헉은 인디언 조가 의사 로빈슨을 살해하는 장면을 보았다.

Tom and Huck saw Injun Joe murder Dr. Robinson.

프로퍼티로 객체 내부 값에 안전하게 접근한다.

class WitnessReport:
    def __init__(self, witness):
        self._witness = witness

    @property
    def witness(self):
        return self._witness

report = WitnessReport("Tom and Huck")
print(report.witness)

The three boys imagined themselves as pirates and enjoyed a free life.

응용 16 · 클래스 메서드

장례식의 반전

장례식 날 죽은 줄 알았던 톰과 친구들이 교회에 갑자기 나타났다.

On the day of the funeral, Tom and his friends suddenly appeared at the church.

클래스 메서드는 클래스 자체의 정보를 이용해 객체를 만들 수 있다.

class ReturnEvent:
    def __init__(self, place, people):
        self.place = place
        self.people = people

    @classmethod
    def from_story(cls):
        return cls("church", ["Tom", "Huck", "Joe"])

event = ReturnEvent.from_story()
print(event.place, event.people)
응용 17 · 정적 메서드

법정의 용기

톰은 무고한 사람이 벌을 받는 것을 견딜 수 없어 법정에서 진실을 말했다.

Tom could not bear seeing an innocent man punished, so he told the truth in court.

정적 메서드는 객체 상태와 관계없는 도구 함수처럼 사용할 수 있다.

class Court:
    @staticmethod
    def judge_truth(statement):
        return "truth" in statement.lower()

statement = "Tom tells the truth about Injun Joe."
print(Court.judge_truth(statement))
심화 18 · 매직 메서드

진짜 범인

진짜 범인이 인디언 조라는 사실이 밝혀지자 그는 재판장에서 도망쳤다.

When Injun Joe was revealed as the real criminal, he escaped from the courtroom.

__eq__를 사용해 두 사건 기록이 같은 범인을 가리키는지 비교한다.

class CrimeRecord:
    def __init__(self, criminal):
        self.criminal = criminal

    def __eq__(self, other):
        return self.criminal == other.criminal

r1 = CrimeRecord("Injun Joe")
r2 = CrimeRecord("Injun Joe")
print(r1 == r2)
심화 19 · 예외 처리

보물 단서

톰과 헉은 인디언 조가 숨긴 보물 이야기를 듣고 탐험을 시작했다.

Tom and Huck heard about treasure hidden by Injun Joe and began their search.

보물 단서가 없을 때 예외를 발생시켜 안전하게 처리한다.

class TreasureMap:
    def __init__(self, clue=None):
        self.clue = clue

    def find_clue(self):
        if self.clue is None:
            raise ValueError("No clue found.")
        return self.clue

try:
    m = TreasureMap("cave")
    print(m.find_clue())
except ValueError as e:
    print(e)
심화 20 · 추상 클래스

동굴 모험

톰과 베키는 소풍 중 거대한 동굴에 들어갔다가 길을 잃었다.

Tom and Becky entered a huge cave during a picnic and became lost.

추상 클래스는 반드시 구현해야 할 메서드의 약속을 만든다.

from abc import ABC, abstractmethod

class Adventure(ABC):
    @abstractmethod
    def solve_problem(self):
        pass

class CaveAdventure(Adventure):
    def solve_problem(self):
        return "Find a safe exit."

adventure = CaveAdventure()
print(adventure.solve_problem())

식량과 촛불이 점점 부족해졌지만 톰은 침착하게 출구를 찾으려 했다.

탈출

톰은 마침내 작은 출구를 찾아 베키와 함께 동굴을 빠져나왔다.

Tom finally found a small exit and escaped from the cave with Becky.

class EscapePlan:

심화 23 · 제너레이터 메서드

인디언 조의 최후

나중에 사람들은 동굴 안에서 죽은 인디언 조를 발견했다.

Later, people found Injun Joe dead inside the cave.

yield를 사용하면 사건을 하나씩 순서대로 꺼낼 수 있다.

class CaveEvents:
    def timeline(self):
        yield "cave sealed"
        yield "people return"
        yield "Injun Joe found"

for event in CaveEvents().timeline():
    print(event)
심화 24 · 데코레이터

보물 발견

톰과 헉은 동굴에서 금화가 가득 담긴 보물 상자를 발견했다.

Tom and Huck found a treasure chest filled with gold coins in the cave.

데코레이터로 메서드 실행 전후에 추가 기능을 붙인다.

def announce(func):
    def wrapper(*args, **kwargs):
        print("Treasure search begins!")
        result = func(*args, **kwargs)
        print("Treasure search ends!")
        return result
    return wrapper

class TreasureHunter:
    @announce
    def find(self):
        return "gold coins"

hunter = TreasureHunter()
print(hunter.find())
심화 25 · OOP 종합 설계

성장과 책임

톰은 장난꾸러기에서 용감하고 책임감 있는 소년으로 성장했다.

Tom grew from a mischievous boy into a brave and responsible young man.

여러 클래스를 함께 설계해 성장 과정을 객체 지향적으로 표현한다.

class Trait:
    def __init__(self, name, score):
        self.name = name
        self.score = score

class CharacterGrowth:
    def __init__(self, name):
        self.name = name
        self.traits = []

    def add_trait(self, trait):
        self.traits.append(trait)

    def summary(self):
        for trait in self.traits:
            print(f"{self.name}'s {trait.name}: {trait.score}")

tom = CharacterGrowth("Tom")
tom.add_trait(Trait("mischief", 50))
tom.add_trait(Trait("courage", 95))
tom.add_trait(Trait("responsibility", 90))
tom.summary()
