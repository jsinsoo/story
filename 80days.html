10. 장면 문장

다른 회원들은 불가능하다고 생각했다.

The other members thought it was impossible.

# 10번 문장: 데코레이터
def travel_frame(func):
    def wrapper():
        print("🌍 80일간의 세계 일주 장면 시작")
        func()
        print("🚂 장면 끝")
    return wrapper

@travel_frame
def show_scene():
    print('다른 회원들은 불가능하다고 생각했다.')
    print("→", 'The other members thought it was impossible.')

show_scene()

70. 장면 문장

포그는 정해진 날짜를 넘겼다고 생각했다.

Fogg thought he had missed the appointed date.

# 70번 문장: 데코레이터
def travel_frame(func):
    def wrapper():
        print("🌍 80일간의 세계 일주 장면 시작")
        func()
        print("🚂 장면 끝")
    return wrapper

@travel_frame
def show_scene():
    print('포그는 정해진 날짜를 넘겼다고 생각했다.')
    print("→", 'Fogg thought he had missed the appointed date.')

show_scene()
