55. 심화 · 데코레이터 팩토리
허클베리 핀의 여행은 자유와 우정, 양심의 중요성을 보여 주었다.
Huckleberry Finn's journey demonstrated the importance of freedom, friendship, and conscience.
학습 포인트: 데코레이터 팩토리
def emphasize(word):
    def decorator(func):
        def wrapper():
            return word + "! " + func()
        return wrapper
    return decorator

@emphasize("중요")
def theme():
    return "자유, 우정, 양심"

print(theme())
56. 심화 · 종합: 데코레이터 + 함수
이 이야기는 미국 문학을 대표하는 성장 소설로 평가받고 있다.
This story is regarded as one of the greatest coming-of-age novels in American literature.
학습 포인트: 종합: 데코레이터 + 함수
def title_box(func):
    def wrapper(title):
        print("=" * 30)
        result = func(title)
        print("=" * 30)
        return result
    return wrapper

@title_box
def evaluate(title):
    return title + "은 대표적인 성장 소설입니다."

print(evaluate("허클베리 핀의 모험"))
