100개 줄거리 문장을 순서대로 빠짐없이 활용하여 파이썬 기초·응용·심화 10단계 문법을 서로 다른 예시 코드로 학습합니다.

10. 피터팬 장면

English
Wendy was curious about the strange boy.

한글 번역
웬디는 그 신기한 소년에 대해 궁금해했다.

# 10번 문장: 데코레이터
def neverland_frame(func):
    def wrapper():
        print("🧚 피터팬 장면 시작")
        func()
        print("🐍 파이썬 학습 완료")
    return wrapper

@neverland_frame
def show_scene():
    print('웬디는 그 신기한 소년에 대해 궁금해했다.')
    print("→", 'Wendy was curious about the strange boy.')

show_scene()

# 100번 문장: 데코레이터
def neverland_frame(func):
    def wrapper():
        print("🧚 피터팬 장면 시작")
        func()
        print("🐍 파이썬 학습 완료")
    return wrapper

@neverland_frame
def show_scene():
    print('피터 팬은 영원히 어린 소년으로 남아 어린 시절의 상상력과 모험의 상징이 되었다.')
    print("→", 'Peter Pan remained forever young, becoming a symbol of childhood imagination and adventure.')

show_scene()
